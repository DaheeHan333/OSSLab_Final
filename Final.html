Bubble Sort

  • Pseudocode - bubble sort
  • BubbleSort(A[], N)
  • for i=1 to N-1
  • for j=1 to N-i
  • if(Arr[j] > Arr[j+1]) then
  • interchange (Arr[j], Arr[j+1]
  • end for
  • end for

Selection Sort

  • Pseudocode - selection sort
  • SelectionSort(A[], N)
  • for i=1 to N-1
  • min = i
  • for j=i+1 to N
  • if A[j] < A[min] then
  • min = j
  • end for
  • if min!=i then interchange A[i] and A[min]
  • end for

Insertion Sort

  • Pseudocode - insertion sort
  • InsertionSort(A)
  • for j=2 to A.length
  • key = A[j]
  • while j > 0 and A[i] > key
  • A[i+1] = A[i]
  • i = i-1
  • A[i+1] = key
  • end for

Merge Sort

  • Pseudocode - merge sort
  • MergeSort(A,p,r)
  • if p < r
  • q = (p+r)/2
  • MergeSort(A,p,q)
  • MergeSort(A,q+1,r
  • Merge(A,p,q,r)
  • Pseudocode - merge sort
  • Merge(A,p,q,r)
  • n1 = q-p+1
  • n2 = r-q
  • let L[1..n1+1] and R[1..n2+1] be new arrays
  • for i=1 to n1
  • L[i] = A[p+i-1]
  • for j=1 to n2
  • R[j] = A[q+j]
  • L[n1+1] = INF
  • i = 1
  • j = 1
  • for k=p to r
  • if L[i] <= R[j]
  • A[k] = L[i]
  • i = i+1
  • else A[k] = R[j]
  • j = j+1

Quick Sort

  • Pseudocode - quick sort
  • QuickSort(A,p,r)
  • if p < r
  • q = PARTITION(A,p,r)
  • QuickSort(A,p,q-1)
  • QuickSort(A,q+1,r)
  • Pseudocode - quick sort
  • Partition(A,p,r)
  • x = A[r]
  • i = p-1
  • for j=p to r-1
  • if A[j] <= x
  • i = i+1
  • exchange A[i] with A[j]
  • exchange A[i+1] with A[r]
  • return i+1

Heap Sort

  • Pseudocode - heap sort
  • HeapSort(A)
  • BuildHeap(A)
  • for i=A.length to 2
  • exchange A[1] with A[i]
  • heapsize = heapsize - 1
  • Heapify(A,1)
  • Pseudocode - heap sort
  • BuildHeap(A)
  • heapsize = A.length
  • for i=floor(length/2) to 1
  • Heapify(A,i)
  • Pseudocode - heap sort
  • Heapify(A,i)
  • le = i.left
  • ri = i.right
  • if (le<=heapsize) and (A[le]>A[i])
  • largest = le
  • else
  • largest = i
  • if (ri=heapsize) and (A[ri] > A[largest])
  • largest = ri
  • if largest != i
  • exchange A[i] with A[largest]
  • Heapify(A, largest)
